# InkShelf Icons

This directory contains icon files for the InkShelf extension.

## Required Icons

The extension requires the following icon sizes:
- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon32.png` - 32x32 pixels (Windows taskbar)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Temporary Setup

For development, you can create simple placeholder icons or use any PNG images with the correct dimensions.

### Quick Placeholder Creation

You can use any image editor or online tool to create simple colored squares:

1. **icon16.png**: 16x16px PNG
2. **icon32.png**: 32x32px PNG
3. **icon48.png**: 48x48px PNG
4. **icon128.png**: 128x128px PNG

### Design Guidelines

For production icons, consider:
- Use a simple, recognizable symbol (e.g., an open book, ink bottle, or markdown "M")
- Ensure good contrast and visibility at small sizes
- Follow Chrome extension design guidelines
- Use transparent background (PNG with alpha channel)
- Keep the design simple and clean

## Color Suggestion

InkShelf brand color: `#0066cc` (blue) on white/transparent background
